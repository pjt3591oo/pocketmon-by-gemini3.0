# Pocketmon Game

A simple Pokemon-style game built with HTML5 Canvas and JavaScript.

## How to Play

1. Open `index.html` in your web browser.
2. **Move**: Use Arrow Keys (⬆️, ⬇️, ⬅️, ➡️).
3. **Explore**: Walk into the **Light Green Grass** to find wild Pokemon.
4. **Battle**:
   - **[Z]**: Attack / Continue dialogue.
   - **[X]**: Run away.
5. **Goal**: Defeat wild Pokemon and survive!

## Features

- Tile-based movement.
- Random encounters in tall grass.
- Turn-based battle system.
- Basic Pokemon stats (HP, Attack, Type).
- Game Over and Restart functionality.
